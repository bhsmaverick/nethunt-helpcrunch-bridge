 Покрокове підключення сервісів

                    

                    

                        
1. Налаштування Webhook в HelpCrunch

                        
Для того щоб HelpCrunch передавав дані про чати та користувачів на цей міст, вам потрібно зареєструвати Webhook URL.

                        
                        

                            
                            

                                
                                
                            

                            Скопіюйте це посилання та вставте його в кабінеті HelpCrunch.
                        


                        
    
                            
  1. Перейдіть до вашого кабінету HelpCrunch.
    2. 
                            
  2. Відкрийте Settings (Налаштування) → Developers (Розробники) → Webhooks.
    3. 
                            
  3. Натисніть Add Webhook (Додати Webhook).
    4. 
                            
  4. Вставте скопійовану вище адресу в поле Payload URL.
    5. 
                            
  5. Виберіть формат передачі даних: JSON.
    6. 
                            
  6. Скопіюйте Webhook Signing Key (якщо хочете підтверджувати підписи) і збережіть його в налаштуваннях цього моста.
    7. 
                            
  7. Позначте події, на які сервіс повинен реагувати:
                                
      
                                    
    • chat.new — Створення нового чату клієнтом (головна подія).
      • 
                                    
    • customer.new — Реєстрація нового користувача.
      • 
                                    
    • message.chat.customer — (За бажанням) Дозволить оновлювати CRM при першому повідомленні.
      • 
                                
    
                            
    8. 
                            
  8. Натисніть Save (Зберегти).
    9. 
                        


                        


                        
2. Налаштування кастомних полів у NetHunt CRM

                        
Щоб інтеграція працювала коректно за всіма каналами пошуку, переконайтеся, що у вашій папці контактів NetHunt створено відповідні поля:

                        
    
                            
  • Телефон та Email — стандартні поля NetHunt, які зазвичай присутні у всіх за замовчуванням.
    • 
                            
  • Telegram — створіть поле типу Text або Link із назвою, що збігається із зазначеною у налаштуваннях моста (за замовчуванням: Telegram).
    • 
                            
  • HelpCrunch Chat Link — створіть текстове поле для запису посилання на поточний діалог клієнта (за замовчуванням: HelpCrunch Chat Link). Це поле буде заповнюватися автоматично, що дозволить відкривати діалог HelpCrunch прямо з карти клієнта в NetHunt CRM.
    • 
                        


                        


                        
3. Автоматичне створення контактів у NetHunt CRM

                        
Якщо за запитом у CRM не знайдено існуючого клієнта (за Email, номером телефону чи Telegram), система автоматично створює нову картку в NetHunt. Для створення картки використовуються такі дані з HelpCrunch:

                        
                        

                            

                                
                                    

                                        Дані з HelpCrunch
                                        Поле в NetHunt CRM
                                        Опис та налаштування
                                    

                                
                                
                                    

                                        Ім'я клієнта (Name)
                                        Name (Стандартне)
                                        Записується у стандартне поле імені картки контакту. Якщо ім'я відсутнє, створюється як "Unknown Customer".
                                    

                                    

                                        Email адреса (Email)
                                        Кастомне поле для Email (За замовчуванням: Email)
                                        Заповнюється значенням з поля емейлу в HelpCrunch. Поле NetHunt для запису налаштовується в параметрах.
                                    

                                    

                                        Номер телефону (Phone)
                                        Кастомне поле для телефону (За замовчуванням: Phone)
                                        Заповнюється номером телефону користувача. Назва поля NetHunt вказується у налаштуваннях.
                                    

                                    

                                        Telegram Username (customData)
                                        Кастомне поле Telegram (За замовчуванням: Telegram)
                                        Отримується із властивості customData (наприклад, telegram) у HelpCrunch. Символ @ видаляється автоматично для чистоти пошуку.
                                    

                                    

                                        Посилання на діалог (Chat URL)
                                        Кастомне поле діалогу (За замовчуванням: HelpCrunch Chat Link)
                                        Записується пряме посилання на діалог в інбоксі HelpCrunch. Створюється відразу під час реєстрації картки (якщо активовано опцію двостороннього зв'язку).
